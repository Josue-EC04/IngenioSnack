# IngenioSnack — Sistema Web de Cafetería Digital

Este proyecto es una plataforma web desarrollada para la cafetería **IngenioSnack** de la UNCP.
Fue construido utilizando metodologías ágiles (Extreme Programming - XP) como parte del curso de Ingeniería de Software.

## Estructura del Repositorio (Semana 10)

El proyecto sigue una estructura estricta orientada al desarrollo ágil y TDD:

- `/src`: Código fuente completo del programa (Frontend en React + Backend en Node.js/Express).
- `/tests`: Pruebas unitarias bajo el enfoque TDD.
- `/docs`: Documentación del sistema.
- `/EVIDENCIAS_TDD`: Capturas del flujo Red-Green-Refactor.
- `BITACORA_PAIR_PROGRAMMING.md`: Registro de las sesiones de trabajo en pareja.
- `REFACTORIZACION.md`: Registro de los code smells solucionados y mejoras de clean code.

## Despliegue (Deploy)

El proyecto está configurado para desplegarse automáticamente en **Render** mediante el archivo `render.yaml` que se encuentra en la raíz del repositorio, el cual apunta correctamente a los servicios ubicados en `/src/frontend` y `/src/backend`.
